Tentu, berikut adalah langkah-langkah untuk membuat shortcut dari awal, mulai dari membuat atau menentukan folder atau file hingga mengganti ikon dan nama shortcut di Windows:

### Langkah 1: Membuat atau Menentukan Folder atau File
1. Buka File Explorer dengan menekan tombol Windows + E.
2. Navigasikan ke lokasi di mana Anda ingin membuat shortcut, misalnya di Desktop.
3. Untuk membuat folder baru, klik kanan di area kosong di File Explorer, pilih "New", lalu pilih "Folder". Atau untuk membuat file baru, Anda bisa membuatnya langsung, misalnya dengan menggunakan aplikasi Office atau Notepad.
4. Beri nama folder atau file sesuai keinginan Anda.

### Langkah 2: Membuat Shortcut
1. Klik kanan pada folder atau file yang ingin Anda buat shortcutnya.
2. Pilih "Send to" di menu konteks yang muncul.
3. Pilih "Desktop (create shortcut)".

### Langkah 3: Mengganti Nama dan Icon Shortcut
1. Di desktop, klik kanan pada shortcut yang baru saja dibuat.
2. Pilih "Properties" dari menu yang muncul.
3. Di jendela Properties, pilih tab "Shortcut".
4. Klik tombol "Change Icon".
5. Pilih ikon yang ingin Anda gunakan dari daftar yang tersedia atau klik "Browse" untuk mencari ikon di lokasi lain.
6. Setelah memilih ikon, klik "OK" untuk menyetujui perubahan.
7. Kembali ke jendela Properties, Anda juga dapat mengubah nama shortcut di bagian "Shortcut name".
8. Setelah selesai, klik "Apply" dan kemudian "OK" untuk menyelesaikan proses.

Dengan mengikuti langkah-langkah di atas, Anda akan dapat membuat shortcut dari awal, menyesuaikan ikon dan nama sesuai dengan preferensi Anda di Windows.
